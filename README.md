# Raketech-form-test-server

NodeJs express server communication with the wp-json api.

# For Development

server is running on http://localhost:4007

```
npm install
```
Run the app
```
node app.js
```

Unit Tests
```
npm run test
```

.env file for development should contain
``` 
HOST=http://localhost
```
