# Raketech-form-test-server

NodeJs express server communication with the wp-json api.

# For Development
```
npm install
```
Run the app
```
node app.js
```

Unit Tests
```
npm run test
```

.env file for development should contain
``` 
HOST=http://localhost
```

# For Production 

.env file for production should contains
```
HOST=http://18.194.254.254
```
